# PDF Scanner & Editor

A premium Flutter application for scanning, editing, and refining PDF documents.

## Key Features

- **Visual Editor**: Draw, add text, and annotate your PDF documents with a rich toolset.
- **Text Refinement**: Extract text using OCR and refine it within our built-in text editor.
- **Secure Saving**: Integrated save confirmation to ensure all your edits are correct before finalizing.
- **Auto-save**: Never lose your progress! The app automatically saves your work if you accidentally navigate away.
- **Signature Support**: Add digital signatures directly to your refined documents.

## Getting Started

To get started with development:

1. Clone the repository.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` onto your device or emulator.

## Build

To build a release APK:
`flutter build apk --release`

The built APK will be located at `build/app/outputs/flutter-apk/app-release.apk`.
